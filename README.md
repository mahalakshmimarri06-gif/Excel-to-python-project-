# Excel-to-python-project-

Amazon India Sales Performance Analysis📋 

Project Overview
As a Junior Data Analyst at Amazon India, this project focuses on analyzing sales transaction data to derive actionable insights for business strategy. 
The analysis covers sales performance across different states, product categories, payment modes, and customer segments to support data-driven 
decision-making for the upcoming quarter.
🛠 Tech Stack
Excel:
Used for creating interactive dashboards, KPI tracking, and data visualization.  
Python (Pandas/Matplotlib/Seaborn):
Used for data cleaning, statistical analysis, and complex visualization tasks. 
📊 Project ScopePart 1: 
Excel DashboardThe dashboard includes:  
KPI Cards: Total Sales, Total Profit, Total Orders, Average Sales, and Average Profit.
Visualizations: Bar charts (Sales by State), Column charts (Profit by Category), Pie charts (Sales by Category), and Line charts (Monthly Sales Trend).
Interactivity: Slicers for State, Category, and Payment Mode.
Part 2: Python Data Analysis
The analysis pipeline includes:  Data Cleaning & Exploration: Loading datasets, checking structure, and generating statistical summaries.
KPI Calculation: Deriving key performance metrics.
Performance Analysis: Evaluating sales trends by geography, category, and sub-category.
Customer & Payment Insights: Identifying top customers and preferred payment methods.
Visualization: Implementing charts using Python libraries to represent data trends.
💡 Key Business Questions Addressed
This project provides data-driven answers to critical questions, including: 
Revenue Drivers: Which categories and sub-categories contribute the most?
Geographic Performance: Which regions lead in sales, and which require improvement?
Customer Behavior: Who are the top 5 customers, and what are their payment preferences?
Profitability: Identification of high-sales, low-profit products and strategic recommendations for management.
